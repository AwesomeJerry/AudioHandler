# AudioHandler
A tiny library for replacing HTML5 &lt;audio>.

## Why
HTML5 &lt;audio> does a terrible job on mobile, especially when playing multiple audios at the same time.

## Usage
```js
AudioHandler.play(file_url);
AudioHandler.stop(file_url);
```
